# アルテマブラスター練習

FF14「絶妖星乱舞」向けの、アルテマブラスター散開位置を練習する非公式ファンメイドページです。

1回目・2回目のケフカ突進から、ケフカの回転方向、サイコロの並び、自分の立ち位置を判断する練習ができます。

## 使い方

### モード

このページには3つのモードがあります。

#### ギミック理解

ギミックの仕組みを確認するためのモードです。

以下の3つをまとめて確認します。

1. ケフカの回転方向
2. サイコロ1番の位置
3. 自分のサイコロ番号の立ち位置

ギミックの考え方を覚えるための確認用です。

#### 位置確認

1回目・2回目の突進を見て、自分の立ち位置をマップ上から選ぶモードです。

実戦練習よりは落ち着いて確認できますが、最終的な散開位置を直接答えるため、ギミック理解より実戦寄りです。

#### 実戦練習

実際の流れに近いタイミングで練習するモードです。

スタートすると3カウント後に突進が始まります。
ピンク色の球がケフカの突進を表します。

6回目の突進で自分のサイコロが表示されます。
その後、時間内に正しい散開位置をマップ上から選びます。

シークバーがなくなると自動で判定されます。
時間切れを待たずに判定したい場合は、実戦練習内の「判定」ボタンを押してください。

## ギミックの考え方

フィールドマーカーは時計回りに以下の順で見ます。

```text
A → 1 → B → 2 → C → 3 → D → 4
```

見るものは主に2つです。

1. 1回目突進の到着地点
2. 1回目から2回目で、ケフカの出現位置が時計回りに進んだか、反時計回りに進んだか

サイコロは、ケフカの進行方向と逆回りに並びます。

### サイコロ1番の位置

サイコロ1番は、1回目突進の到着地点そのものではありません。

1回目の到着地点から、サイコロが並ぶ方向に見た次のマーカーとの中間に置きます。

たとえば、1回目突進が `2 → 4`、2回目突進が `C → A` の場合、

* 1回目の出現位置：2
* 2回目の出現位置：C
* `2 → C` は時計回り
* ケフカは時計回り
* サイコロは逆の反時計回り
* 1回目の到着地点は4
* 4から反時計回り側の次マーカーはD

したがって、サイコロ1番は `D-4間` です。

## 散開位置

サイコロの立ち位置は、マーカーそのものではなく、隣り合うマーカーの中間です。

```text
A-1間
1-B間
B-2間
2-C間
C-3間
3-D間
D-4間
4-A間
```

自分のサイコロ番号に対応する位置を選びます。

## 番号設定

番号は以下の2種類から選べます。

* ランダム
* 自分で選ぶ

「自分で選ぶ」では、1〜8のうち練習したい番号を固定できます。
苦手な番号を重点的に練習したいときに使ってください。

## 実戦練習の操作

### スタート

3カウント後に問題が始まります。

### 一時停止

実戦練習中の進行を一時停止します。
再開すると停止したところから続きます。

### 判定

時間切れを待たずに、その時点の回答を判定します。

### 次の問題

新しい問題を開始します。

### 同じ問題をもう一度

現在と同じ突進パターン・同じ番号でやり直します。

### 1問前に戻る

直前の問題に戻ります。

### 連続モード

デフォルトの進行モードです。

判定後、正解・不正解を表示したあと、自動で次の問題に進みます。

### 1問ごとモード

判定後に自動で次へ進まず、手動で次の問題や同じ問題のやり直しを選べます。
